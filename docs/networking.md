# Networking 

## CCNA Day 1 - Network Devices

## concepts learned

- hosts
- switch
- router 
- Firewall 
- Server 
- Client

## Summary

In this first video, I learned what types of devices exist on a network and what the function of each one is. I also understood that hosts 
communicate with each other through devices like switches and routers, depending on whether they are on the same network or diferent networks
(A host is any device that sends or receives data on a network, A switch connects hosts within the same network (LAN)
 and a router connects different networks (internet))
