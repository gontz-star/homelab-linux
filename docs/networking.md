## CCNA Day 3 - TCP/IP Model

## concepts learned


- TCP/IP model
- Encapsulation
- Decapsulation
- Application layer
- Transport layer
- Internet layer
- Network Access layer
- TCP
- UDP
- IP
- Ethernet

### Summary

In this video, I learned how data travels through the TCP/IP model. Each layer has a specific role in preparing, transmitting, and receiving data.
I also learned about encapsulation and decapsulation, where each layer adds or removes its own header as data moves between devices.

### Key Takeaways

- The TCP/IP model is the networking model used on the Internet.
- Each layer has a specific responsibility.
- Data is encapsulated before being transmitted and decapsulated when received.
- TCP provides reliable communication, while UDP prioritizes speed.
